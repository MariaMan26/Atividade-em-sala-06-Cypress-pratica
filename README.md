# Atividade em Sala 06 — Testes com Cypress

Este projeto consiste na validação completa de uma página de **Cadastro de Usuário**, desenvolvida em **HTML**, **CSS** e **JavaScript**, utilizando o framework **Cypress** para automação de testes end-to-end.

A atividade foi proposta em sala, onde recebemos o código da página e tivemos como objetivo criar uma suíte de testes capaz de validar **todos os elementos visuais** e **todas as regras de validação** do formulário.

---

## Objetivo da Atividade

- Testar os elementos estáticos da página (títulos, labels, inputs, botões).
- Validar o comportamento dos campos do formulário.
- Garantir que todas as regras implementadas no JavaScript sejam corretamente atendidas:
  - Campos obrigatórios.
  - Validação simples de e-mail.
  - Senha com no mínimo 8 caracteres.
  - Comparação entre senha e confirmação de senha.
- Verificar a exibição de mensagens de erro e sucesso.
